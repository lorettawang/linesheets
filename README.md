# **Linesheets** 

Linesheets is a B2B wholesale e-commerce platform that allows wholesalers to build your linesheets, share them to your buyers, and capture orders.

## **Technologies Used**

* CSS3
* JavaScript
* Ruby on Rails
* VS Code
* Bash
* GitHub
